Public deals will be listed in public market. In both cases, users will connect the wallet they are using to the website and will choose which token they would like to make a deal for. After that, they will able to set either a set price or auction.
Private deals are only eligible by the link that generated by the deal creator. In both cases, users will connect the wallet they are using to the website and will choose which token they would like to make a deal for. After that, they will able to set either a set price or auction.
